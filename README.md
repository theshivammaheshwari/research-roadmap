# Research Roadmap

A smart literature review tool for PhD scholars and researchers.

## Features

- Search papers by topic and year range
- Auto-categorize papers (Survey, Research, Benchmark, Dataset, Explainability)
- Reading methodology guide (3-pass method)
- Export results to CSV (opens in Excel)
- Dark / Light mode
- Direct links: DOI, Sci-Hub, Open Access PDF
- Fully responsive

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **OpenAlex API** (free, open scholarly data)
- **Vercel** deployment

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Push this repo to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Deploy automatically

## API

Powered by [OpenAlex](https://openalex.org) — a free, open index of scholarly works with 250M+ papers.
